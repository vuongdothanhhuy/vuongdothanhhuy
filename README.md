# Hi, I'm Huy 👋

I'm a **senior frontend engineer** and a **Roman Catholic catechist**.
I care about clarity, responsibility, and truth — in faith ✝️, in teaching 📖, and in software 💻.

## Engineering work 🧑‍💻

I build **enterprise-scale web applications** with a focus on systems
that are predictable, maintainable, and easy to reason about.

My primary stack is **Angular 14+ with NgRx** for complex state management,
**TypeScript** throughout, and data integration via **GraphQL (Apollo Client)
and REST APIs**. I also work across the stack when needed — **Node.js,
Next.js, and AWS Lambda** are in the toolkit.

Most of my recent experience is in **financial compliance (RegTech)**:
KYC identity verification pipelines, real-time AML/CTF transaction monitoring,
and enterprise-grade data visualisation using **Vis.js, D3.js, and Plotly.js** 📊.
Before that, several years in **e-commerce** for EU clients —
with **GDPR-first system design** baked in from day one 🔐.

I'm an active practitioner of **AI-augmented development** 🤖 —
using **Claude Code, Gemini CLI, and GitHub Copilot** as force multipliers,
guided by senior architectural judgment.
The goal: accelerated delivery *without* sacrificing code quality or test coverage.

## Teaching & service ⛪

Outside of work, I serve as a **catechist**,
dedicating my weekends to teaching children,
mentoring youth, and supporting church administration.

Pedagogy shapes how I think. If an idea cannot be explained simply —
especially to children — it is not yet truly understood.
That principle strongly influences how I write code, documentation,
and technical decisions.

## Curiosity & systems thinking 🔍

I enjoy learning how systems work at a fundamental level, across different domains:
